# AIエージェント向け指示書 (AI Agent Instructions)

このプロジェクトへようこそ！このワークスペースはAgentic RAGを用いたテックブログ執筆環境です。

まずはプロジェクトの全体像を把握するため、以下のドキュメントを読み込んでください。

1. 👉 **[README.md](./README.md)** (アーキテクチャやフォルダ構成の起点)
2. 👉 **[LLM_WIKI_GUIDELINES.md](./LLM_WIKI_GUIDELINES.md)** (自律的永続ナレッジベース「llm-wiki」の運用ルール)

このガイドラインには、あなたがユーザーの裏側で密かにWiki（`tobi/qmd`）を検索し、生データを取り込んでエンティティを更新することで「Agentic RAG」機能を提供する手順が記載されています。このガイドラインを絶対に読み飛ばさないでください。
